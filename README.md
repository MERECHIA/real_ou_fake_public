# Fake ou Real? (AI Experiemce)

Projeto para uma feira de inovação em inteligência artificial

## Descrição

O projeto "Fake ou real?" é ume experiência interatica onde o usuário precisa idenificar se uma imagem foi gerada por inteligência artificial ou se é real.

A proposta é conscientizar os visitantes sobre o avanço das IAs generativas e os desafios de distinguir conteúdos reais de articiais.

## Funcionalidades 

- Exibição de imagens (reais e geradas por IA)
- Sistema de escolha: "REAL" ou "FAKE"
- Feedback imediaato ao usuário
- Sistema de pontuação 
- Explicações sobre cada imagem qunado erra
- Integração com câmera para detectar presença

## Tecnologias utilizadas

- HTML
- CSS
- JavaScript
- Web APIs (gemini-3.1-flash-lite)
- ML5
- tinyfacedetector 

## Uso da Câmera

- Detectar a presença do usuário
- Iniciar a interação automaticamente
- Reiniciar automaticamente perante saida do usúario da frente da tela
- Tornar a experiência mais dinâmica

## Estrutura de Branches

- 'main' -> versão final do projeto
- 'dev' -> integração das funcionalidades
- 'features/' -> desenvolvimetno de novas funcionalidades
- 'hotfix/' -> correções rápidas
